******************************************************
Casino de Bender - Juegos de azar y mujerzuelas
******************************************************

Para compilar el codigo que implementa el casino, solamente basta con correr el comando "make" en este directorio. 

Para jugar en el casino, después de compilar el código, ingresar una serie de personas según su tipo, por ejemplo:

./a.out TEDTED

Una vez que se ha probado, se puede correr el comando "make clean" para borrar archivos binarios.
